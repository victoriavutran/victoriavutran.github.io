Data

The dataset contained 12,575 rows with several missing values across 11 columns:

  • Transaction ID - String

  • Customer ID - String

  • Category - String

  • Item - String

  • Price Per Unit - Decimal

  • Quantity - Integer

  • Total Spent - Decimal

  • Payment Method - String

  • Location - String

  • Transaction Date - Date

  • Discount Applied - Boolean

Python libraries pandas, numpy, and matplotlib were used to clean and validate the data.

Data Wrangling

Before solving for null values, the column Total Spent was validated with expression (Price Per Unit)(Quantity).


Solve for Column Price Per Unit:

Price Per Unit = Total Spent / Quantity knowing that Total Spent = (Price Per Unit)(Quantity).


Solve for Column Discount Applied:

A lookup dictionary containing each item’s regular (non-discounted) unit price was created. If the calculated price per unit was lower than the standard price, the transaction was marked as discounted.

Example: Regular price of Item 1: $3

Customer purchased 10 units for $10 → $1 per unit

Since $1 < $3, the transaction was flagged as having a discount applied.


Solve for Column Item:

A reverse lookup dictionary was then created to fill the null values of column Item. If an item’s price matched a known unit price, the correct Item name was assigned.
